# Mars Rover with AI

This project implements the [Mars Rover Kata](https://codingdojo.org/kata/mars-rover/) using Test-Driven Development (TDD).

## Important Notes

- **AI-Driven Development**: All changes in this project have been done 100% by an AI agent.
- **No Python Experience**: This project was created by someone with no prior Python knowledge, relying entirely on AI assistance.
- **TDD Approach**: The entire project was developed using Test-Driven Development (red-green-refactor cycle).
- **Development Process**: For detailed steps taken during development, check the git commit log: `git log --oneline`
